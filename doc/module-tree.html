Module Hierarchy

  • generate_predictions: Loads a trained model and outputs predictions based on input feature files.
  • learner: Module with many functions to use for easily creating a scikit-learn learner
  • metrics: This module contains a bunch of evaluation metrics that can be used to evaluate the performance of learners.
  • print_model_weights: Simple script for printing out model weights.
  • run_ablation: Runs an ablation study, removing one feature file at a time.
  • run_experiment: Runs a bunch of scikit-learn jobs in parallel on the cluster given a config file.
